# Time Tracker Pro

**Track your team's productivity with ease!**

Created by **Karas Barsoum**

---

## Features

- ✅ **Clock IN/OUT** - Start and stop timers for each team member
- ✅ **Live Timer** - See real-time elapsed time
- ✅ **Multiple People** - Track unlimited team members
- ✅ **Time Records** - View all clock in/out history
- ✅ **Admin Mode** - Edit records and adjust hours (password protected)
- ✅ **PDF Export** - Generate professional reports with total hours per person
- ✅ **Data Persistence** - All data saved locally

---

## How to Use

### Web Version
1. Double-click `start-server.bat`
2. Open http://localhost:8080 in your browser

### Desktop App
1. Install Node.js from https://nodejs.org
2. Double-click `BUILD_APP.bat` to create the .exe
3. Find `Time Tracker Pro.exe` in the `dist` folder

### Deploy Online (Free)
1. Create a GitHub account
2. Double-click `DEPLOY_ONLINE.bat` and follow instructions
